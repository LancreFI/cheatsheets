Disk image information
qemu-img info disk.img

Create raw disk image
qemu-img create disk.img 80G

Start with disk only
 qemu-system-x86_64 -hda disk.img

Starting the emulator with CD-image
qemu-system-x86_64 -hda disk.img -cdrom cd.iso -boot d

Networking disabled
 qemu-system-x86_64 -hda disk.img -net none

Set memory size
qemu-system-x86_64 -m 6G -hda disk.img

Enable KVM
 qemu-system-x86_64 -hda disk.img -enable-kvm

Assing more than one CPU
 qemu-system-x86_64 -hda disk.img -smp <CPU COUNT>

Share folder from host
sudo apt install samba; qemu-system-x86_64 -hda disk.img -net nic -net user,smb=/tmp/bar

 Ctrl-Alt-+ == decrese screen size
 Ctrl-Alt-u == restore screen size
 Ctrl-Alt-1 == show target display
 Ctrl-Alt-2 == show monitor
 Ctrl-Alt-3 == show serial
